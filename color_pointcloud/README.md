# color_pointcloud

A simple package to colorize a LiDAR point cloud with RGB information from the corresponding camera images. The accurate 3 x 4 projection matrix from the LiDAR to the camera is assumed to be given.

### Packages used
* pcl_ros
* opencv
* cv_bridge
* message_filters
